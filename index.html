Iron Bank of Falkreath
Connect your Google Sheet once. Everyone uses the same HTML file and URL.
Step 1 — Create a Google Sheet
Go to sheets.google.com and create a blank spreadsheet.
Step 2 — Open Apps Script
In your sheet go to Extensions → Apps Script. Delete any existing code, paste in the iron-bank-falkreath.gs file, and save.
Step 3 — Deploy
Deploy → New deployment · Type: Web app · Execute as: Me · Who has access: Anyone · Deploy → authorize → copy the Web app URL.
Step 4 — Paste URL below
Everyone opens this HTML file and enters the same URL once. It's saved in their browser after that.
